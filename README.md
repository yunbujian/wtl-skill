# wtl — 写作高效学习任何领域 2.0

用写作驱动学习的 AI 共创 Skill。

核心信念：写作不是学完之后做的事，写作本身就是学习。

## 三个入口

| 你的状态 | 说这句话 |
|---|---|
| 不知道从哪里开始 | 「我不知道从何入手」 |
| 写到中途卡住了 | 「我卡住了」 |
| 想把内容发出去 | 「我想发出去」 |

说任何一句，都可以开始。

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/yunbujian/wtl-skill/main/install.sh | bash
```

安装后在 Claude Code 里输入：

```
/wtl
```

## 这个 Skill 做什么

- **🔥 启动**：不知道写什么时，AI 帮你找到那个说不清楚的地方，给你一个开始写的入口
- **🔄 突破**：写到卡住时，AI 引入对立视角，帮你从局部最优里拉出来
- **🚀 发表**：写完后，AI 帮你整理成可以发出去的格式（公众号、小红书、网页等）

每次反馈都会标注对应的原理课内容，方便回溯理论。

## 关于这个课程

这个 Skill 背后是「写作高效学习任何领域」课程的 2.0 版本。

1.0 有原理，2.0 有入口。你不需要先学完原理再开始——说一句话，直接进入。
